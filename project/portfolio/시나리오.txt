메인 화면 (차트)
==> 50개 음악 리스트 보여 준 후 
==> 유튜브 버튼 눌렀을 때 해당 노래의 유튜브로 새창을 띄어 이동 

==> 회원가입 페이지
==> 중복확인 버튼 한개라도 수행 X 등록 X
==> 중복확인 버튼 모두 수행시 등록 O
==> 회원가입 완료 시 자동 로그인

==> 프로필 페이지
==> 내 정보 수정 버튼 클릭 시 수정 화면 전환
==> 이메일 변경 ==> 정규식 검사 및 중복확인 
==> 비밀번호 변경 ==> 정규식 검사 후 변경
==> 버그 : 닉네임 및 이메일 수정 한 후 수정 완료 버튼 누르면 수정이 됨.

==> 로그인 페이지
==> 아이디 저장 및 자동 로그인 기능
==> 카카오 로그인 

==> 아이디/비밀번호 찾기 페이지
==> 아이디 찾기 ==>
이름과 이메일을 입력 후 아이디 찾기 버튼을 누르면 아이디 알림창에 띄움
==> 비밀번호 찾기 ==>
아이디와 이메일을 입력 후 비밀번호 찾기 누르면
임시 비밀번호 알림창에 띄움.

==> 커뮤니티 페이지
==> 로그인 X ==> 읽기 만
==> 로그인 O ==> CRUD 가능
==> 관리자 로그인 ==> 커뮤니티 수정 가능

==> 관리자 페이지
==> 회원관리 ==> 닉네임 변경, 회원 삭제
==> 권한관리 ==> 권한 추가 및 박탈
==> 차트 리셋
